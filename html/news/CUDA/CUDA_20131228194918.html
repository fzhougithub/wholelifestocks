Barracuda Launches Secure Remote Access for Amazon Web Services Cloud Customers

Barracuda NG Firewall Now Available in the Amazon Web Services Marketplace

Barracuda Networks December 23, 2013 10:15 AM
0 shares
Content preferences

    CAMPBELL, Calif., Dec. 23, 2013 /PRNewswire/ -- Barracuda Networks Inc. (CUDA), a leading provider of cloud-connected security and storage solutions, today announced that the Barracuda NG Firewall is now available in the Amazon Web Services Marketplace. The Barracuda NG Firewall for Amazon Web Services (AWS) allows customers to protect and access their server infrastructures deployed in the AWS cloud.

    (Logo: http://photos.prnewswire.com/prnh/20131113/SF16521LOGO)

    "The Barracuda NG Firewall offers AWS customers comprehensive features to securely deploy in public cloud environments," said Blair Hankins, VP Engineering, Barracuda. "This expanded offering extends remote network access to augment the other Barracuda security offerings available on the Amazon AWS cloud platform."

    The Barracuda NG Firewall provides VPN and security-enhanced remote network access between customer networks and the AWS cloud platform, including highlights such as:

    • Granular control over applications, allowing administrators to define rules for forwarding data traffic using the best respective transmission channels based on type of application, user, content, time of day, and geographical location. The Barracuda NG Firewall allows organizations to prioritize traffic by limiting or restricting access to non-business-related applications and network traffic, even when encrypted.
    • Enhanced network security with the Barracuda NG Firewall Intrusion Detection and Prevention System (IDS/IPS), providing complete and comprehensive real-time network protection against a broad range of network threats, vulnerabilities, exploits, and exposures in operating systems, applications, and databases preventing network attacks such as SQL injections and arbitrary code executions.
    • Advanced site-to-site and client-to-site VPN capabilities, using both SSL and IPsec protocols to ensure remote users can easily and securely access network resources without complex client configuration and management.

    Pricing and Availability

    The Barracuda NG Firewall in the AWS Marketplace is available today as a Bring Your Own License service. User licenses can be purchased through traditional channels starting at U.S. list price $1,599 USD for a one-year subscription, and customers can then use that license to provision the application in AWS. For more information, please visit: https://aws.amazon.com/marketplace/pp/B00H8CB6HY/ref=_ptnr_pr_ngfirewall.
